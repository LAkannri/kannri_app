🌟 エンカンAIってなに？

あなたの代わりに、申請フォームの入力を全自動でやってくれる相棒ロボットです。

これまで「1件あたり5分」かかっていたキャリア申請業務を、エンカンAIがまるごと引き受けます。 あなたは 確認するだけ でOK。 この手順書では、エンカンAIをあなたのパソコンで使えるようにするまでを案内します。

⏱ 所要時間の目安 初回セットアップ：合計15〜20分（待ち時間が多いです）
2回目以降の起動：3秒（ダブルクリックだけ！）

🎒 準備するもの

作業を始める前に、これらが揃っているか確認してください。

  • インターネットにつながったパソコン（Windows または Mac）
  • 管理者から共有された secrets.toml の中身（3つの接続キー）
  • 30分くらいの作業時間（ほぼ待ち時間です）
⚠️ secrets.tomlがまだ無い方へ 管理者にお願いして、3つの接続キー（SUPABASE_URL、SUPABASE_KEY、GEMINI_API_KEY）を教えてもらってください。 Slackなどの社内チャットで安全に共有してもらいましょう。
STEP 1

🐍 Pythonをインストールする

エンカンAIを動かすための「土台」になるソフトです。

  1. Pythonの公式サイトを開く

    https://www.python.org/downloads/ を開いて、「Download Python 3.x.x」の黄色いボタンをクリックします。

  2. ダウンロードしたファイルを開く

    画面下に出てきた「python-3.x.x-amd64.exe」というファイルをダブルクリックします。

  3. ⚠️ 一番大事なところ！

    インストール画面の 「Add Python to PATH」 の左にあるチェックボックスを 必ずONにしてください。これを忘れるとエンカンAIが起動しません。

    （スクショを差し込む場所）
    「Add Python to PATH」のチェックを入れた状態
  4. 「Install Now」をクリック

    あとは画面の指示に従って、「Close」が出るまで待ちます。3〜5分くらいかかります。

❌ もし「Add Python to PATH」のチェックを忘れたら Pythonをアンインストールして、もう一度インストールし直してください。チェックを入れた状態でないとエンカンAIは動きません。
  1. Pythonの公式サイトを開く

    https://www.python.org/downloads/ を開いて、「Download Python 3.x.x」の黄色いボタンをクリックします。

  2. ダウンロードしたファイルを開く

    「python-3.x.x-macos.pkg」というファイルをダブルクリックします。

  3. インストーラーの指示に従う

    「続ける」「同意する」「インストール」を順番にクリックしていきます。途中でMacのパスワードを求められたら入力してください。

    （スクショを差し込む場所）
    Macのインストーラー画面
  4. 「閉じる」が出たら完了！

    3〜5分くらいで終わります。

STEP 2

📦 エンカンAIをダウンロードする

GitHubというサイトから、エンカンAIのフォルダ一式をダウンロードします。

  1. 管理者から共有されたGitHubのリンクを開く

    例：https://github.com/会社名/enkan-ai のようなURLです。

  2. 緑色の「Code」ボタンをクリック

    ページの右上あたりにある緑色のボタンを押すと、メニューが出てきます。

    （スクショを差し込む場所）
    GitHubの「Code」ボタンの位置
  3. 「Download ZIP」をクリック

    メニューの一番下にある「Download ZIP」をクリックすると、ZIPファイルがダウンロードされます。

  4. ZIPファイルを解凍する

    Windowsの方：ZIPファイルを右クリック →「すべて展開」
    Macの方：ZIPファイルをダブルクリック

  5. 分かりやすい場所に置く

    解凍されたフォルダを、デスクトップなど分かりやすい場所に移動させましょう。フォルダ名は enkan-ai など好きな名前に変えてOKです。

💡 フォルダの中身を確認 解凍したフォルダを開いて、こんなファイルがあればOKです：
app.py / robot.py / 起動_Windows.bat / 起動_Mac.command / .streamlitフォルダ など
STEP 3

🔑 接続キー（secrets.toml）を設置する

エンカンAIがデータベースなどに接続するための「鍵」を置きます。

  1. 「.streamlit」というフォルダを開く

    STEP 2で解凍したフォルダの中にある .streamlit フォルダ（先頭にドットが付いてます）をダブルクリックして開きます。

  2. 「secrets.toml.example」を見つける

    その中に secrets.toml.example というファイルがあります。これがテンプレートです。

  3. ファイル名から「.example」を消す

    ファイル名を右クリック →「名前の変更」を選んで、末尾の .example を削除します。
    変更前：secrets.toml.example
    変更後：secrets.toml

  4. ファイルをメモ帳（テキストエディット）で開く

    Windowsの方：ファイルを右クリック →「プログラムから開く」→「メモ帳」
    Macの方：ファイルを右クリック →「このアプリケーションで開く」→「テキストエディット」

  5. 3つのキーを書き換える

    「ここに〇〇を入れる」と書かれている部分を、管理者からもらった本物のキーに置き換えます。 ダブルクォーテーション（"）の中だけを書き換えてください。

    SUPABASE_URL = "https://xxxxx.supabase.co"
SUPABASE_KEY = "eyJhbGc..."
GEMINI_API_KEY = "AIzaSy..."
  6. 保存して閉じる

    Ctrl+S（MacはCommand+S）で保存します。

🔒 secrets.tomlの取り扱いについて このファイルには重要な接続キーが書かれています。 絶対にメールやチャットで他の人に送ったり、GitHubにアップしたりしないでください。
STEP 4

🚀 起動ボタンをダブルクリック！

いよいよエンカンAIを起動します。ここから先は自動で進みます。

  1. 「起動_Windows.bat」をダブルクリック

    エンカンAIのフォルダの中にある 起動_Windows.bat をダブルクリックします。

  2. 「Windowsによってデバイスが保護されました」と出たら

    「詳細情報」をクリック →「実行」をクリックしてください。
    ※これは Windows が「初めて見るファイルだから心配だよ」と教えてくれているだけです。安全なファイルなので大丈夫です。

    （スクショを差し込む場所）
    Windows SmartScreenの警告画面
  3. 黒い画面が出てきます

    初回は「必要な部品をダウンロード中...」と表示されます。
    5〜10分ほど待ってください。お茶でも飲みながらどうぞ ☕

  4. ブラウザが自動で開いたら成功！

    エンカンAIの画面が表示されたら、セットアップは完了です。お疲れ様でした！

  1. 「起動_Mac.command」をダブルクリック

    エンカンAIのフォルダの中にある 起動_Mac.command をダブルクリックします。

  2. 初回だけ「開けません」と言われたとき

    ① 「起動_Mac.command」を 右クリック →「開く」を選ぶ
    ② 確認画面が出たらもう一度「開く」をクリック
    ※2回目以降は普通にダブルクリックでOKです

    （スクショを差し込む場所）
    Macで右クリック→開くの操作
  3. ターミナル（黒い画面）が出てきます

    初回は「必要な部品をダウンロード中...」と表示されます。
    5〜10分ほど待ってください。お茶でも飲みながらどうぞ ☕

  4. ブラウザが自動で開いたら成功！

    エンカンAIの画面が表示されたら、セットアップは完了です。お疲れ様でした！

🆘 困ったときは（Q&A）

よくあるトラブルと、その解決方法をまとめました。

黒い画面に「Pythonが見つかりません」と出る

STEP 1のPythonインストールで「Add Python to PATH」のチェックを忘れた可能性が高いです。 Pythonを一度アンインストールして、もう一度インストールし直してください。 その際、必ず「Add Python to PATH」にチェックを入れてください。

「接続キーの設定ファイルがありません！」と出る

STEP 3を飛ばしてしまった可能性があります。 .streamlit フォルダの中に secrets.toml（末尾が .example ではなく .toml）があるか確認してください。

初回セットアップで「インストールに失敗しました」と出る

インターネット接続が不安定な可能性があります。 Wi-Fiが安定した場所で、もう一度起動ファイルをダブルクリックしてみてください。 会社のネットワーク制限に引っかかっている場合は、管理者に相談してください。

ブラウザが自動で開かない

黒い画面に http://localhost:8501 のようなURLが表示されているはずです。 そのURLをコピーして、自分でブラウザのアドレスバーに貼り付けてアクセスしてください。

録画ボタンを押してもブラウザが開かない

初回セットアップで Playwright のブラウザインストールが終わっていない可能性があります。 一度黒い画面を閉じて、フォルダ内の .setup_done というファイルを削除してから、 もう一度起動ファイルをダブルクリックしてください。セットアップが最初からやり直されます。

エンカンAIを終わらせたい

黒い画面で Ctrl + C（Macは Control + C）を押してください。 その後、黒い画面を閉じてOKです。

新しいバージョンに更新したい

管理者から「更新してください」と連絡があったら、もう一度GitHubから最新のZIPをダウンロードして、 古いフォルダと差し替えてください。 その際、自分の secrets.toml は新しいフォルダにコピーするのを忘れずに！

その他のトラブル

上記で解決しない場合は、黒い画面に出ているエラーメッセージをスクショして、管理者に相談してください。

🚫 やってはいけないこと

ここだけは守ってください。事故を防ぐためのお約束です。

  • secrets.toml を他の人にメールやチャットで送る
  • secrets.toml をGitHubにアップロードする
  • エンカンAIが動いている最中に黒い画面を×で閉じる（→ Ctrl+Cで止めましょう）
  • app.py や robot.py の中身を勝手に書き換える（管理者に相談してから）
  • 会社のSFAデータを社外のサービスに勝手にアップロードする
